# 🏦 Credit Risk Prediction

## 📌 Objective

Develop a machine learning model to predict whether a loan applicant is likely to default on a loan based on demographic and financial information.

## 🛠️ Approach

* Performed data cleaning and handled missing values.
* Conducted Exploratory Data Analysis (EDA) using visualizations of income, loan amount, education, and other key features.
* Encoded categorical variables and prepared the dataset for modeling.
* Trained and evaluated classification models, including **Logistic Regression** and **Decision Tree Classifier**.
* Assessed model performance using **Accuracy Score** and **Confusion Matrix**.

## 📊 Results & Insights

* Successfully built a binary classification model for credit risk prediction.
* Identified important factors influencing loan approval and default risk, such as **Credit History**, **Applicant Income**, and **Loan Amount**.
* The trained models achieved satisfactory prediction performance and demonstrated the effectiveness of machine learning techniques in assessing loan risk.

**Tools & Technologies:** Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook.
